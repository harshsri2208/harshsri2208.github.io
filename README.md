# Harsh Srivastava | Software Engineer

A high-energy, dark-mode Neo-Brutalist digital portfolio featuring an integrated AI assistant.

## 🚀 Tech Stack
- **Framework**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS (Custom Dark Neo-Brutalist Theme)
- **Animations**: Motion (Framer Motion)
- **AI Engine**: Google Gemini 1.5 Flash (`@google/genai`)

## ✨ Key Features
- **Interactive AI Assistant**: A custom-tuned, comic-book style chatbot embedded securely in the UI.
- **Dark Neo-Brutalist Design**: Pitch black background (`#111111`), stark white borders, high-contrast colorful panels, and **Sky Blue** (`#38BDF8`) drop shadows.
- **Automated CI/CD**: Auto-deploys to GitHub Pages via Git Actions.
- **Serverless Feedback**: Functional contact form powered by Formspree.

## 🛠️ Setup & Local Dev

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Setup Gemini API**: Get a key from [Google AI Studio](https://aistudio.google.com/app/apikey), then create a `.env` file:
   ```txt
   GEMINI_API_KEY=your_api_key_here
   ```
3. **Run locally**:
   ```bash
   npm run dev
   ```

> **Deployment Note**: For GitHub Pages, ensure `GEMINI_API_KEY` is added to your repository's **GitHub Secrets** (`Settings -> Secrets and variables -> Actions`).
